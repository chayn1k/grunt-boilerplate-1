Why use boilerplate

  • Boilerplate scaffolds out a new application, writing your Grunt configuration and pulling in relevant Grunt tasks and Bower dependencies that you might need for your build.
  • Grunt is used to build, preview and test your project, thanks to help from tasks curated by the grunt-contrib.
  • Bower is used for dependency management, so that you no longer have to manually download and manage your scripts.
  • grunt-modernizr is a Modernizr builder for your project. It will crawl your project for Modernizr test references and save out a minified, uglified, customized version using only the tests you've used in your JavaScript or CSS.
  • bootstrap 3. Automatically downloaded to the latest version. You can freely customize it with the pre-compile less version.
  • html5shiv and respond.js preinstalled.
  • jquery modern (2+) and legacy (1.10) preinstalled.
  • grunt-cssmin. All your CSS will be minimized and compressed with cleancss.
  • grunt-less. All your less imports, variables and mixins will be compiled and pre-pended to your main CSS.
  • grunt-uglify. All your JavaScript files will be automatically concatenated and pre-pended to your main.js file and compressed with uglify.
  • grunt-stencil. A Grunt templating plugin to generate static HTML files from given pages, templates and partials written in either pure HTML, doT.js, or markdown. Wiki
  • grunt-imagemin. Minify images using imagemin. Comes bundled with the following optimizers: gifsicle Compress GIF images, jpegtran Compress JPEG images, optipng Compress PNG images, svgo Compress SVG images modern (2+) and legacy (1.10) preinstalled.
  • grunt-connect and grunt-open. Start a connect web server for test and development. And with grunt-open you'll launch your favourite browser.
  • grunt-watch. All changes made to your files are watched and will launch relative tasks.

Installation

Install Node.js and Git and run all the following commands from console:

  • Install Grunt globally
    • npm install -g grunt-cli
  • Install Bower globally
    • npm install -g bower
  • Fix of Bower issue on windows 7
    • git config --global url."https://".insteadOf git://

For adding Git to Windows 7 Path:

  • Right-Click on My Computer
  • Click Advanced System Settings link from the left side column
  • Click Environment Variables in the bottom of the window
  • Then under System Variables look for the path variable and click edit
  • Add the pwd to git's bin and cmd at the end of the string like this:
    ;C:\Program Files (x86)\Git\bin;C:\Program Files (x86)\Git\cmd

Usage

After installation unzip Boilerplate git or clone git, from console run:
grunt prepare and grunt development

Run grunt help for more commands.
